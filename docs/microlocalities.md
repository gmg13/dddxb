# Microlocalities

The unit of analysis. A **microlocality** = a Dubai community / sub-community (e.g. Al Jaddaf, Business Bay, JVC).

## To define here (as data is adopted)

- **Canonical name list** — the authoritative set of microlocalities we rank.
- **Mapping rules** — how raw source labels (DLD `area`/`community`, listing-site neighbourhood names) normalize to a canonical microlocality. Source naming is inconsistent; this mapping is where that's reconciled.
- **Geometry** — boundary polygons per microlocality for choropleths and spatial joins.
- **Aliases** — known alternate spellings/labels per microlocality.

Until populated, treat the cleaned source `community` field as a provisional microlocality key.
